ThingSpeak API pour objets connectés

Internet se transforme progressivement en un réseau étendu, appelé « Internet des objets », reliant tous les objets devenus connectables.
ThingSpeak est un des maillons de l’internet des objets. C’est une Plate-forme de collecte et de stockage de données avec traitements mathématiques.

Ses trois fonctions principales sont

  • récupérer et stocker les données des objets ;
  • analyser et visualiser les données avec MATLAB ;
  • déclencher des actions lorsque certaines conditions sont remplies.

    • Les objets/capteurs communiquent directement avec la plate-forme ThingSpeak par des requêtes HTTP. Un système de clés est présent pour assurer la sécurité des données, qui sont ensuite stockées dans une base relationnelle. L’arrivée d’une donnée peut être l’élément déclencheur d’une action : le lancement d’un script Matlab qui fait des opérations mathématiques sur les données par exemple.

    Les applications accèdent aux données via des API (Application Programming Interface) afin de les afficher sous forme de graphe et de développer des services spécifiques.

    L'élément principal de ThingSpeak est le "Channel". Un channel enregistre les données envoyées vers ThingSpeak et comprend les éléments suivants :

    • 8 champs pour stocker n'importe quel type de données ;
    • 3 champs de localisation, latitude, longitude et élévation ;
    • d'autres champs metadata.
